Etiquetas semánticas:

Las etiquetas en HTML son código prediseñado con el fin de realizar un orden en la Web y están compuestas por ciertos estándares, como lo son:

  • "<", usado para abrir una nueva etiqueta..
  • Una palabra o carácter que determina qué etiqueta se está escribiendo.
  • Cualquier número de atributos HTML que se quiera usar, escritos de la forma nombre="valor"
  • ">", usado para cerrar una nueva etiqueta.

Listas:

Las listas en HTML nos permite crear conjuntos de elementos en forma de lista dentro de una página, todos los cuales irán precedidos, generalmente, por un guión o número.
se forman de la siguiente forma:



<ul>
<li>Primer elemento</li>
<li>Segundo elemento</li>
<li>tercer elemento</li>
</ul>

Existen multiples formas de realizar listas, a continucion te mostramos en una tabla como se realizan



Etiqueta Función
<ul> Es la etiqueta fundamental de apertura, las listas, a partir de estas las etiquetas hijos se formaran
<li> Forma una lista con el punto inicial
<ol> Las listas ordenadas comienzan con esta etiqueta y el resultado es una lista ordenada por números
<dl> Viene de los términos ingleses “Definiton list” y nos indica que dentro de ella, entre ella y su cierre, va a ir una definición
<dt> Viene de los términos “Definition term” y dentro de ella irá el término que vamos a definir. dentro de esta iría el título de la definición
<dd> Viene de los términos “Definition description” y nos dice que dentro de ésta irá la definición.

Ejemplo Función
